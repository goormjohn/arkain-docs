# May 7, 2026

## **🌟 Precision in Every Detail.**

This update is all about finer control and sharper results. Agent Permissions now respond the moment you change them, and Snap stays locked onto your original intent from start to finish. Combined with a refreshed Template experience, every improvement is designed to put more precision in your hands and less friction in your way.

### Highlights

**⚡ Real-time Agent Permissions**\
Tool permission updates now take effect immediately, even during an active response. If you need to revoke a tool's access mid-execution, your changes apply instantly—allowing the Agent to adapt to new settings without waiting for the next request or needing a fresh start.



**🎯 Enhanced Snap Precision**\
Snap now delivers higher generation quality by maintaining your original intent as a continuous reference. This ensures every execution stays more accurately aligned with your goals.



**📋 Refined Template Experience**\
The Template List layout now features a new "My Templates" button and a repositioned "Official" toggle for better accessibility. Additionally, the default sort order is now set to "Recent" to help you discover a wider variety of fresh project starters.
